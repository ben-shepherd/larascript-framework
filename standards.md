# Coding Standards

## Naming Conventions

### Casing
- **Preferred casing:** camelCase

### File and Folder Naming
- All folders and files should be lowercase
- Exception: Domain folders should be capitalized

## Import Standards
- All imports should be absolute, utilizing `@src` or `@core`

## Examples

```
├── controllers/
├── models/
├── domains/
│   ├── User/
│   └── Movies/
└── utils/
└── helpers.js
```
